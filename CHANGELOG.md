# Changelog

All notable changes are recorded here. The format follows Keep a Changelog,
and releases follow Semantic Versioning.

## [Unreleased]

## [1.1.0] - 2026-07-21

### Added

- Enum-validated finding filters for `severity` and `status`.
- Severity statistics endpoint with total and zero-filled category counts.
- Unit and integration coverage for filter combinations and invalid values.
- API examples for reporting queries.

## [1.0.0] - 2026-07-21

### Added

- FastAPI health endpoint and OpenAPI documentation.
- Validated vulnerability-finding domain model.
- SQLite repository and complete CRUD API.
- Unit and integration tests with branch coverage above 80%.
- GitHub Actions and equivalent GitLab CI/CD pipelines.
- Ruff, Bandit, pip-audit, detect-secrets, and versioned Git hooks.
- Engineering documentation and traceable GitHub issues/PRs.

## [0.1.0] - 2026-07-21

### Added

- Initial repository bootstrap.
